# Design Specification: Remove All Emojis

This specification outlines the systematic removal of all visual graphical emojis across all HTML/Jinja template views to maintain a clean, professional, non-informal enterprise aesthetic.

---

## 1. Targets and Conversions

### Brand Names & Headers
- `HikerHub 🏔️` -> `HikerHub`
- `Eco-Hikes 🏔️` -> `Eco-Hikes`
- `Trek Staff 🏔️` -> `Trek Staff`
- `Trekker 🏕️` -> `Trekker`
- `Welcome back, ... 🥾` -> `Welcome back, ...`

### Badges & Status Indicators
- `Past 🕰️` -> `Past`
- `Active 🏃‍♂️` -> `Active`
- `Future 📅` -> `Future`
- `Easy 🟢` / `Moderate 🟡` / `Hard 🔴` -> `Easy` / `Moderate` / `Hard`
- `Promote to Admin 👑` -> `Promote to Admin`

### Metadata Icons
- `⛰️ Max:` -> `Max:`
- `📏 Len:` -> `Length:`
- `💰 Price:` -> `Price:`
- `⏱️` -> (Remove)
- `📍` -> (Remove)
- `🛡️` -> (Remove)
- `⭐` / `🌟` -> (Remove star symbols, replace star rating with text "Rating:")
